# LoadSync 0.3.1

**Released:** March 09, 2026

## Features

- Link to changelog on advanced tab of help page
- Add support for 12 port PDU model - HZA125BAD1

## Bug Fixes

- Fix some cards not loading data on summary page
- Add error handling and reconnect callbacks for NATS connection
- Add NATSUnexpectedEOF to ignored errors in main application
- Added default state update for resolved alarms during migration
- Fix miner error messages not showing in advanced miner tab
- Ensure warning symbol shows up when a miner has an error
