# LoadSync 0.5.0

**Released:** May 13, 2026

## Features

- Add update V2 API and update the web UI
- Update fleet manager telemetry handler
- Add manifold temperature to the engine page
- Add individual PDUs to the PDU graphs page
- Implement get_alarm_values method in controllers and update AlarmManager
- Improved load shedding logic and recovery mechanisms in PowerManager
- Improve port labelling to match physical port labels per phase
- Add dark mode to the local UI with a toggle

## Bug Fixes

- Remove decimal references and use built in float
- Update telemetry service
- QR code styling in fleet help page
- Fix humidity parsing loading an invalid max value
- Remove border from selected sidebar nav items

## Refactor

- Miners table now uses DataTables
