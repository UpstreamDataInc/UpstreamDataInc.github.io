# LoadSync 0.6.1

**Released:** June 23, 2026

## Features

- Add miner count to graph (#246)
- Add 4000 W power limit alarm to PDU ports (#247)
- Add MinerWatchdog to monitor and reboot miners based on hashrate degradation (#248)
- Add clear success/failure states for the settings pages (#253)
- Update to latest asic-rs (0.7.0) (#261)

## Bug Fixes

- Improve text selection for copy button in fleet help section
- Remove set_fan_speed event spamming logs in 'Auto Fan' mode.
- Increase miner data save frequency
- Update hashrate on miner graph to support TH/s and PH/s based on value (#256)
